# Prompt Creator

Create prompts that are clear, constrained, testable, and tool-aware.

## Guidelines

- State role and objective explicitly.
- Define inputs, outputs, constraints, and success criteria.
- Include tool-specific behavior only when the tool exists.
- Prefer short rules over vague personality text.
- Add examples only when they reduce ambiguity.
- Remove contradictions and outdated platform references.
