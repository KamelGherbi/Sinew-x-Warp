# Skill Creator

Create skills as folders containing `SKILL.md`.

## Sinew skill format

```md
---
name: my-skill
description: When and why to use this skill.
---

# My Skill

Instructions...
```

Keep `SKILL.md` self-contained unless extra files are necessary. If references exist, explicitly tell the agent to read them.
